## 📌 About This Project

**Wing Bank Clone** is a **Flutter-based mobile application** that replicates the user interface and navigation flow of the official Wing banking app.

This project focuses on mastering modern UI/UX design, responsive layouts, state management, and smooth animations — providing a realistic foundation for future fintech or digital payment applications.

---

## 🚀 Key Features

* 📱 Pixel-accurate UI replication of Wing's mobile interface
* 🔄 Smooth page transitions & bottom navigation flow
* 💳 Mock transaction history & account dashboard
* 🌙 Dark/Light theme with adaptive styling
* 📱 Fully cross-platform (Android, iOS, Web, Desktop)
* 🎨 Custom reusable widgets & micro-interactions

---

## 🛠️ Tech Stack

* 💙 **Flutter (Dart)**
* 🎯 **Material Design 3 & Custom Theming**
* 🧭 **Navigation**: `[GoRouter / AutoRoute / Navigator 2.0]`
* 📊 **State Management**: `[Riverpod / Provider / BLoC / GetX]`
* 🗃️ **Local Storage**: `[Shared Preferences / Hive / SQLite]`
* 🧪 **Testing**: `flutter test`, `flutter analyze`

## ⚙️ Getting Started

### 1. Install Flutter

👉 [https://docs.flutter.dev/get-started/install](https://docs.flutter.dev/get-started/install)

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
flutter run
```
